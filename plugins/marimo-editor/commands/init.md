# Initialize Marimo Project

Install marimo and related dependencies using uv.

## Command

Run the following command:

```bash
uv add marimo altair vegafusion vl-convert-python nbformat
```

## Dependencies

- **marimo**: Reactive notebook framework
- **altair**: Declarative statistical visualization
- **vegafusion**: Accelerated Vega-Lite rendering
- **vl-convert-python**: Vega-Lite format conversion
- **nbformat**: Jupyter notebook format support
